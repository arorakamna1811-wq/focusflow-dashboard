# 🌟 FocusFlow Dashboard

A modern and aesthetic productivity dashboard designed using pure HTML & CSS.
This project showcases a clean UI with glassmorphism effects and a structured layout for daily productivity.

---

## 🚀 Features

* 🧠 Task Management Section
* ⏱ Pomodoro Timer UI
* 💬 Daily Motivation Quote
* 🎨 Glassmorphism Design
* ✨ Smooth Hover Animations

---

## 🛠 Tech Stack

* HTML5
* CSS3 (Flexbox + Grid)

---

## 📂 Project Structure

```
focusflow-dashboard/
│── index.html
│── style.css
```

## 🎯 Purpose

This project is built to:

* Practice modern UI design
* Improve CSS layout skills
* Create a portfolio-ready frontend project
